Install Redop directly when you already have a Bun project, or scaffold a starter when you want a ready-to-run server.

## Install the package

```sh
bun add @useagents/redop zod
```

Redop is Bun-first, so the default examples and starter flow assume you will run the server with Bun.

## Scaffold a starter project

If you want a ready-to-run project instead of wiring the first files yourself:

```sh
bun create redop-app my-redop-app
```

That interactive flow can generate either an HTTP or stdio starter and can add deploy presets for hosted HTTP projects.

## Which path should you choose?

- Install `@useagents/redop` directly if you already know the shape you want.
- Use `create-redop-app` if you want a starter app, generated files, and deployment presets.
